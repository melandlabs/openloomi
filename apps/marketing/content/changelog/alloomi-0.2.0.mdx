## Alloomi 0.2.0 - Dec 23, 2025

This Alloomi update proudly unveils a suite of core enhancement features designed to streamline workflows and boost productivity. Leveraging the new UI/UX, along with upgraded capabilities for chat, insights analysis, auto-replies, and additional integrations, existing customers and partners can enjoy a more efficient user experience and superior application outcomes.

### 🔔 Notable Changes

#### 🎨 UI/UX Upgrade

Following this update, Alloomi has fully upgraded its UI rendering engine and interaction logic, placing greater emphasis on the two core functions: Understanding and Integration. Non-core features are organized into secondary entrances, making the workflow more concise and intuitive, and helping you stay focused on core tasks while using Alloomi.

#### 💬 Unified Chat Workspace Upgrade

The Unified Chat Workspace now enables seamless integration with various connected tools. The AI Guidance feature has been optimized to deliver more accurate suggestions and a smoother navigation experience. Quota reminders are more intuitive, helping teams manage usage limits effectively and eliminating workflow disruptions caused by insufficient quotas.

#### 🧠 Insights & Memory System Enhancements

The Insights & Memory System undergoes a major upgrade: Multi-source summaries refresh significantly faster, with precise labeling of responsible persons and urgency levels to help you quickly pinpoint key information and task execution priorities. The optional Memory API supports comprehensive long-term context storage, enabling individuals/teams to accumulate core historical data assets and make more well-informed decisions.

#### 🔀 Custom Group Management

Alloomi now supports custom grouping of core assets including chats, tasks, contacts, and insights. You can create groups based on custom dimensions (e.g., project phases, client types, team roles) and flexibly classify and manage data. This feature breaks the limitations of fixed classification logic, making information organization more personalized and efficient, and helping you quickly locate the content you need.

#### 👥 People Management

We have launched a dedicated People relationship management module that integrates contact information from multiple connected tools (Slack, Gmail, Teams, etc.) to build a unified digital profile for each person. It automatically records interaction history, custom tags, key background information (e.g., cooperation projects, preference notes), and interaction frequency. This enables you to maintain customer/partner relationships more systematically and retrieve relevant context in seconds during communication, enhancing the quality of collaboration and service.

#### 💡 Quick Action

Powered by upgraded AI algorithms, Alloomi now provides intelligent recommended actions based on real-time dialogue content, task priorities, and historical behavior patterns. For example: automatically suggesting follow-up reminders for pending tasks, recommending document creation for summarized insights, or prompting collaboration notifications for cross-team projects. These personalized suggestions reduce decision-making costs and effectively drive task implementation.

#### ⚙️ Task Automation

The new automation task execution capability allows you to configure trigger rules (e.g., receiving messages with specific keywords, approaching task deadlines, updating contact tags) and preset corresponding automated operations (e.g., sending follow-up emails, creating to-do items in task management tools, updating project status, pushing notifications to team members). This feature liberates you from repetitive manual work, improves the automation level of daily workflows, and ensures timely response to key events.
