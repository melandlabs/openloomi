## Alloomi 0.3.0 - Mar 18, 2026

This Alloomi update proudly unveils a suite of core enhancement features designed to streamline workflows and boost productivity. Leveraging the new UI/UX, along with upgraded capabilities for chat, insights analysis, auto-replies, and additional integrations, existing customers and partners can enjoy a more efficient user experience and superior application outcomes.

### 🔔 Notable Changes

#### 🎨 Onboarding Redesign

Alloomi has completely reimagined the onboarding experience with a focus on personalization and engagement. The new onboarding flow introduces AI Avatar functionality, allowing users to create a personalized digital presence from the start. The personalization system has been refactored to deliver more relevant and tailored experiences. Internationalization support has been expanded to better serve our global user base. Additionally, the 8-card popover interaction has been optimized for smoother user engagement and clearer navigation paths.

#### 🧠 Alloomi Soul Personalized Settings

The Alloomi Soul feature brings advanced personalization to a new level. Users can now choose from a variety of Soul presets or create custom prompts to define their AI assistant's personality and behavior. This enhanced personalization system ensures that Alloomi adapts to individual preferences and work styles, delivering more contextually relevant responses and a more personalized assistant experience.

#### ⚡ Skills Ecosystem

Alloomi introduces a comprehensive Skills ecosystem that significantly expands its capabilities. The platform now includes specialized skill sets for Finance and Legal domains, enabling users to handle industry-specific tasks with greater precision. Skill search now supports internationalization, making it easier for global teams to find and utilize relevant skills. The Skills directory has been expanded to include over 235 external skills, providing extensive functionality across various professional domains.

#### 🔗 Platform Integrations

This release brings significant expansion in platform connectivity:

- **X (Twitter) OAuth**: Native X authentication with integrated credit system for seamless access
- **QQ Bot Platform**: New platform support for QQ messaging ecosystem
- **Brave Search Integration**: Enhanced search capabilities with privacy-focused browsing
- **Native PDF API**: Improved document handling with native PDF processing support
- **Manual Platform Display**: Alloomi logo now displays for manually connected platforms, improving brand consistency

#### ⏰ Scheduled Jobs & Cross-Platform Notifications

The scheduled jobs functionality has been significantly enhanced to support more complex automation scenarios. Users can now configure time-based triggers with greater flexibility. Cross-platform notification support ensures that important updates reach users regardless of their preferred communication channels, improving response times and workflow continuity.

#### 🖥️ Desktop Application Improvements

The desktop application has received substantial backend enhancements:

- **Keychain Secrets Management**: Secure credential storage using system keychain integration
- **Encrypted Environment Variables**: Sensitive configuration passed securely via stdin
- **Removed Bundled Node.js**: Reduced application size and improved performance
- **Application Splash Screen**: Enhanced startup experience with branded loading screen

#### 📝 Registration Page Optimization

The registration flow has been redesigned with an engaging hero section that better communicates Alloomi's value proposition. The optimized design improves first impressions and increases conversion rates while maintaining a clean, intuitive user experience.

#### 💡 UI/UX Enhancements

Several interface improvements have been made based on user feedback:

- **Chat Sidebar Improvements**: Enhanced navigation and organization for chat conversations
- **Insight Sidebar Optimization**: Better information hierarchy and faster access to key insights
- **Focused Insight Enhancements**: Improved visualization and interaction for detailed analysis views
